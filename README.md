# oh-my-posh-theme

Just my command line theme

![WindowsTerminal_260704](https://github.com/user-attachments/assets/d2687d68-36d2-482b-ad4a-df35af4993f5)

## Features

- Easy to copy path
- Git
- Exit code
- Execution time
- Sys time
- Prompt symbol (# for admin, $ for user)
- [rprompt](https://ohmyposh.dev/docs/configuration/block#type) user info in windows accent color

### Languages

- Go
- Kotlin
- Java
- Dart
- Zig
- Rust
- Python
- Node
- Lua
- PHP
